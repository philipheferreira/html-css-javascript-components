O que é o innerHTML?

De forma direta: innerHTML é uma propriedade dos elementos HTML que permite você obter 
ou alterar todo o conteúdo HTML que está dentro de um determinado elemento.

Pense em um elemento HTML como uma caixa. A tag em si (por exemplo, <div>) 
é a caixa, e tudo o que está entre a tag de abertura (<div>) e a de fechamento 
(</div>) é o conteúdo da caixa. O innerHTML é o comando que te dá acesso para 
"olhar" ou "trocar" o que está dentro dessa caixa.


Como Funciona: Os Dois Lados da Moeda
O innerHTML tem duas funções principais: ler e escrever.

1. Ler o Conteúdo (Getter)
Você pode usar innerHTML para pegar o conteúdo HTML de um elemento como 
uma string. Isso inclui não apenas o texto, mas também todas as tags HTML 
filhas que estiverem lá dentro.


2. Alterar o Conteúdo (Setter)
Você também pode usar innerHTML para substituir completamente o conteúdo 
de um elemento por uma nova string de HTML. O navegador irá interpretar essa 
string e a renderizar como elementos HTML reais na página.


 O Grande Alerta: Risco de Segurança (XSS)
Esta é a parte mais importante sobre innerHTML. Usá-lo de forma 
imprudente pode abrir uma brecha de segurança grave chamada 
Cross-Site Scripting (XSS).

Por quê? Porque o navegador executa qualquer código JavaScript que 
encontrar dentro da string que você atribui ao innerHTML.
