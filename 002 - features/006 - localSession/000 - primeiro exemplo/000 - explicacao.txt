# A Explicação Simples (A Analogia)
Se o localStorage é um caderno de anotações permanente, o sessionStorage é como um bloco de rascunho ou uma folha de papel 
em branco que você usa para uma tarefa específica.

* Você abre uma aba do navegador e começa a trabalhar. Enquanto essa aba estiver aberta, você pode escrever e ler coisas 
no seu bloco de rascunho.
* Se você abrir uma nova aba do mesmo site, ela terá um bloco de rascunho completamente diferente, em branco.
* No momento em que você fecha a aba, o bloco de rascunho é jogado fora. Tudo que estava nele se perde para sempre.

# A Explicação Técnica
O sessionStorage é outra forma de armazenamento de dados do lado do cliente (client-side), assim como o localStorage. 
Ele também armazena dados em pares de "chave" e "valor".

A grande e principal diferença está no ciclo de vida dos dados.

#Características Principais do sessionStorage

1.Temporário / Por Sessão: Esta é a sua característica definidora. Os dados salvos no sessionStorage duram apenas enquanto 
a aba ou janela do navegador estiver aberta. Quando o usuário fecha a aba, todos os dados são apagados automaticamente. Se 
o usuário recarregar a página (F5), os dados são mantidos, pois a sessão da aba continua.
2.Escopo (Origem e Aba): Este é outro ponto crucial. O sessionStorage é isolado não apenas por origem (domínio), mas também 
por aba/janela.
* Se você tiver duas abas do meusite.com abertas, cada uma terá seu próprio sessionStorage independente. O que você salvar 
em uma aba não será visível na outra.
* Isso é útil para situações onde você não quer que dados de uma tarefa "vazem" para outra.
3.Métodos Idênticos ao localStorage: A forma de usar é exatamente a mesma. Você usa os mesmos quatro métodos:
* sessionStorage.setItem(chave, valor)
* sessionStorage.getItem(chave)
* sessionStorage.removeItem(chave)
* sessionStorage.clear()
4. Armazena Apenas Strings: Assim como o localStorage, ele só salva strings. Para objetos e arrays, você precisa usar 
JSON.stringify() para salvar e JSON.parse() para ler.
5. Capacidade: A capacidade também é de cerca de 5 MB por origem.
